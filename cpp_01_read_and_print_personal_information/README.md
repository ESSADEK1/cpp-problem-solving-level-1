# C++ Problem 01 – Read and Print Personal Information

## 📌 Description

This program demonstrates basic input and output operations in C++.

The user enters:

- Name
- Age
- City
- Country
- Monthly Salary
- Gender
- Marital Status

The program then calculates the yearly salary and displays all the information in a formatted output.

- User Input (`cin`, `getline`)
